# Instalación
Se utilizó pnpm como gestor de dependencias. Para instalarlo, ejecuta el siguiente comando: `npm install -g pnpm`. Luego, navega al directorio del proyecto y ejecuta `pnpm install` para instalar las dependencias.
## Configuración para el desarrollo
Opcional, pero recomendado: instala Prettier con el siguiente comando: `pnpm add -D prettier prettier-plugin-tailwindcss`. Luego, configura Prettier para que formatee automáticamente al guardar el archivo. En IntelliJ IDEA, accede a la opción Actions on Save y selecciona Run Prettier, además de Optimize Imports.
# Ejecutar proyecto
Para visualizar la aplicación en el entorno de producción, se recomienda ejecutar el siguiente comando: `pnpm prod`. Este comando generará una versión optimizada y la iniciará en el puerto 5173.

Si preferís ejecutar la aplicación en modo desarrollo, utiliza `pnpm dev`.
